# Live Sign Language to Text and Speech
## Installation
- Clone the repository
- Install python
- Install the requirements
```bash
C:/C/Z/Live-SignLang-TextNSpeech/venv_sls/Scripts/python.exe -m pip install -r requirements.txt
```
## Running Application
- There was already an existing model. But you can still further train it using the {non-main}.py file
- Run the application using this comand in your terminal
```bash
C:/C/Z/Live-SignLang-TextNSpeech/venv_sls/Scripts/python.exe c:/C/Z/Live-SignLang-TextNSpeech/main.py
```
